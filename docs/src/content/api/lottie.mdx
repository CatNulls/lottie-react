## Options

**`data: string | any`**
- **Required**
- The animation that will be used.
- If a string is passed, it will be treated as a path to the animation file.
- If anything other than string is passed, it will be treated as animation data.
- The animation will automatically reload when the path or data changes.

**`listeners?: LottieAnimationEvents`**
- _Optional_
- An object in which listeners, of type `AnimationEventHandler | null`, can be optionally defined for the following event keys:
  - `onComplete`
  - `onLoopComplete`
  - `onEnterFrame`
  - `onSegmentStart`
  - `onConfigReady`
  - `onDataReady`
  - `onDataFailed`
  - `onLoadedImages`
  - `onDOMLoaded`
  - `onDestroy`

**`containerProps?: React.HTMLProps<HTMLDivElement>`**
- _Optional_
- Object containing the properties that you want to pass to the `<div></div>` used as a container for the animation.
- **Attention:** The `ref` property will have no effect because it is used/overwritten internally by `lottie-react`.
